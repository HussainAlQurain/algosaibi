# AHALGOSAIBI Static Website

This is a static website for **Ahmad Hamad ALGOSAIBI & BROS CO.**

## Language Support

- English page: `index.html`
- Arabic RTL page: `ar/index.html`
- Language switch buttons are included in the header.

## Hosting (Bluehost cPanel)

Upload all files/folders in this directory to your `public_html` folder:

- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/images/*`

No database, build process, or runtime server is required.

## Replace Placeholder Media

Place your official brand media in `assets/images/` using these names:

- `logo.png` (already added)
- `hero-bg.jpg` (optional future hero image)
- `milestone-aramco.jpg`
- `milestone-pepsi.jpg`
- `milestone-crown.jpg`
- `milestone-insulation.jpg`
- `milestone-chamber.jpg`
- `milestone-jotun.jpg`
- `company-construction.jpg` (optional)
- `company-manufacturing.jpg` (optional)

If you want to add a hero video later:

- Place `hero-video.mp4` in `assets/images/`
- Add a `<video>` section in `index.html` hero area.
